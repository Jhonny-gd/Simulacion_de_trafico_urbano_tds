# Frontend - Simulacion de Trafico Urbano

Interfaz web construida con Vue 3, Vuetify, Axios y Vite.

## Ejecutar

```powershell
npm install
npm run dev
```

URL local:

```text
http://127.0.0.1:5173
```

## Build

```powershell
npm run build
```

## Backend esperado

```text
http://127.0.0.1:8000
```
